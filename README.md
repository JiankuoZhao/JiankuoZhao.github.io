# Jiankuo Zhao Academic Homepage

This repository contains the GitHub Pages academic homepage for Jiankuo Zhao, adapted from the PRISM personal website template.

## Site URL

- Repository name: `JiankuoZhao.github.io`
- Homepage URL: `https://JiankuoZhao.github.io`

## Local Development

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Build

```bash
npm run build
```

The static site is exported to the `out/` directory.

## Content Files

- `content/config.toml`: name, title, institution, social links, navigation
- `content/bio.md`: homepage biography
- `content/education.md`: education history
- `content/publications.bib`: publication list in BibTeX format
- `public/avatar.png`: 512 x 512 homepage avatar
- `public/papers/brief.png`: STAvatar publication card image
- `PUBLISH_STEPS.md`: GitHub Pages publishing guide

The CV page is currently disabled because no CV PDF is provided.
